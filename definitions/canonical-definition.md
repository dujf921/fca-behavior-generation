# Canonical Public Definition | CAL — Causal Architecture of Life

CAL — Causal Architecture of Life is a causal framework for describing how living systems generate, stabilize, repeat, interrupt, and update behavior under continuity constraints.

Its public analysis object is the causal relation between:

- the system's current state;
- historically formed available structures;
- current internal and external information;
- continuity-directed constraints;
- observable output;
- reality feedback and structural update.

Observable behavior is an analysis entry and a system output. It is not accepted as its own root cause.

CAL does not posit an independent choice operator outside the causal chain. What appears externally as selection is treated as the causal resolution produced by the system's active constraints and available structures.

## Minimum Public Chain

Current State  
→ Available Structures  
→ Continuity-Directed Constraint  
→ Causal Selection  
→ Observable Output  
→ Reality Feedback  
→ Structural Update

## Public Purpose

The public CAL definition provides a stable identity, analysis object, minimum chain, terminology boundary, and citation source.

It does not disclose the complete variable system, equations, weighting relationships, inference procedures, implementation logic, or executable Runtime model.

## Invalid Reductions

CAL is not:

- a personality typology;
- a psychological labeling system;
- a motivation theory;
- a moral evaluation framework;
- a fixed brain-network or hormone correspondence table;
- a clinical diagnosis system;
- a chatbot wrapper;
- a public specification of the AlitaOS Runtime.
