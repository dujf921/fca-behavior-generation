# 标准公开定义 | CAL — 生命因果架构

CAL — Causal Architecture of Life（生命因果架构）是用于描述生命系统如何在存续连续性约束下生成、稳定、重复、中断并更新行为的因果架构。

其公开分析对象是以下要素之间的因果关系：

- 系统当前状态；
- 历史形成的可达结构；
- 当前内部与外部信息；
- 存续方向约束；
- 可观察输出；
- 现实反馈与结构更新。

可观察行为既是分析入口，也是系统输出，不能被直接当作自身的根因。

CAL 不设定一个位于因果链之外的独立选择操作器。外部观察到的“选择”，被定义为当前系统约束与可达结构共同作用后形成的因果收敛结果。

## 最小公开链路

当前状态  
→ 可达结构  
→ 存续方向约束  
→ 因果选择  
→ 可观察输出  
→ 现实反馈  
→ 结构更新

## 公开目的

CAL 公开定义只负责提供稳定的理论身份、分析对象、最小链路、术语边界与引用来源。

它不披露完整变量体系、公式、权重关系、推断程序、实现逻辑或可执行 Runtime 模型。

## 无效压缩

CAL 不属于：

- 人格分类；
- 心理标签系统；
- 动机理论；
- 道德评价框架；
- 固定脑网络或激素对应表；
- 临床诊断系统；
- 聊天壳；
- AlitaOS Runtime 的公开实现规格。
