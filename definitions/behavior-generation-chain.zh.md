# 最小公开因果链 | CAL

当前状态  
→ 可达结构  
→ 存续方向约束  
→ 因果选择  
→ 可观察输出  
→ 现实反馈  
→ 结构更新

## 1. 当前状态

生命系统以当前配置进入每一轮因果循环。该配置受到身体条件、既往历史、当前情境与新输入信息共同影响。

本公开定义不规定状态如何测量、编码或评分。

## 2. 可达结构

只有从当前状态能够进入的历史形成结构或当前可生成结构，才能参与本轮行为生成。

本公开定义不披露结构提取、表示、调用或匹配程序。

## 3. 存续方向约束

当前生效条件会对可达结构施加方向，使其中一部分结构更符合系统维持存续连续性的要求。

它属于因果方向约束，不是道德偏好、主观目标或已公开的定量函数。

## 4. 因果选择

当前状态、可达结构、现实信息与存续方向约束共同作用后，收敛出当前占主导位置的运行路径。

这里的“选择”只表示因果收敛，不引入因果链之外的独立操作器。

## 5. 可观察输出

占主导位置的路径形成可观察的行动、不行动、重复、中断或状态转换。

输出是因果回溯证据，不是最终因果解释。

## 6. 现实反馈

输出产生的现实结果向系统返回新信息，并改变后续循环的生成条件。

## 7. 结构更新

反馈可能使后续可达结构被维持、减弱、增强、连接或重组。

本公开定义不披露更新公式、权重、触发条件或实现逻辑。

## 边界

该链路只是 CAL 的最小公开拓扑，不是可直接执行的完整 CAL 模型。不得通过猜测补全未公开变量与程序。
