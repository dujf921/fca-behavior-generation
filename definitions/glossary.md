# CAL Public Glossary

| Term | Canonical Public Meaning | Boundary |
| --- | --- | --- |
| CAL | Causal Architecture of Life; a causal framework for behavior generation and update in living systems. | Not a personality typology or Runtime implementation specification. |
| Continuity Constraint | A system-level constraint under which behavior generation remains directed toward maintaining continuity of the living system. | No public equation, score, or parameterization is defined. |
| Current State | The system configuration entering a causal cycle. | No public measurement or encoding procedure is defined. |
| Available Structure | A historically formed or currently constructible structure reachable from the current state. | No public extraction, storage, retrieval, or matching method is defined. |
| Continuity-Directed Constraint | The directional effect of active conditions on which available structures can dominate. | Not a moral preference or conscious intention. |
| Causal Selection | The causal resolution through which one currently reachable route becomes dominant. | Not an independent free-will operator outside the causal chain. |
| Observable Output | Action, non-action, repetition, interruption, or state transition produced by the dominant route. | Analysis entry, not root cause. |
| Reality Feedback | Information returned by the real consequences of output. | Not equivalent to awareness or verbal explanation alone. |
| Structural Update | A change in later reachability or organization caused by feedback. | No public update equation, weight, or trigger rule is defined. |
| Public Definition Layer | The identity, minimum topology, terminology, boundaries, and citation source published in this repository. | Does not contain the complete CAL theory or executable model. |
| Protected Theory Layer | Non-public causal variables, relations, formalizations, and inference rules. | Not disclosed in this repository. |
| Runtime Layer | The private implementation that may operationalize CAL using data, memory, agents, and system logic. | Not disclosed or specified by the public definition source. |
