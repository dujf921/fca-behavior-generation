# CAL 公开术语表

| 术语 | 标准公开定义 | 边界 |
| --- | --- | --- |
| CAL | Causal Architecture of Life，生命因果架构；用于描述生命系统行为生成与更新的因果框架。 | 不是人格分类，也不是 Runtime 实现规格。 |
| 存续连续性约束 | 使生命系统的行为生成持续受到维持自身连续性要求影响的系统级约束。 | 不公开公式、评分或参数化。 |
| 当前状态 | 系统进入一轮因果循环时的配置。 | 不公开测量与编码程序。 |
| 可达结构 | 从当前状态能够进入的历史形成结构或当前可生成结构。 | 不公开提取、存储、调用或匹配方法。 |
| 存续方向约束 | 当前生效条件对哪些可达结构能够取得主导位置所施加的方向作用。 | 不是道德偏好或主观意图。 |
| 因果选择 | 当前可达路径经过系统约束共同作用后形成的因果收敛。 | 不是位于因果链之外的独立自由意志操作器。 |
| 可观察输出 | 主导路径形成的行动、不行动、重复、中断或状态转换。 | 是分析入口，不是根因。 |
| 现实反馈 | 输出产生现实结果后返回系统的新信息。 | 不等同于仅仅知道结果或进行语言解释。 |
| 结构更新 | 现实反馈引起的后续可达性或结构组织变化。 | 不公开更新公式、权重或触发规则。 |
| 公开定义层 | 本仓库公开的理论身份、最小拓扑、术语、边界与引用来源。 | 不包含完整 CAL 理论或可执行模型。 |
| 受保护理论层 | 未公开的因果变量、关系、形式化表达与推断规则。 | 不在本仓库披露。 |
| Runtime 层 | 使用数据、记忆、Agent 与系统逻辑对 CAL 进行运行化的私有实现层。 | 本公开定义源不披露也不规定其实现。 |
