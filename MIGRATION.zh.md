# 迁移指南 | 历史 FCA → CAL

本文件定义历史 FCA 材料向 CAL — Causal Architecture of Life（生命因果架构）的公开名称与解释迁移关系。

## 迁移表

| 历史术语 | 现行公开术语 | 状态 |
| --- | --- | --- |
| Human Behavior FCA | CAL — Causal Architecture of Life | 历史名称已替换 |
| Functional Causal Architecture | Causal Architecture of Life | 历史全称已替换 |
| Functional Causal Architecture for Human Behavior Generation | CAL 公开因果架构 | 历史范围已替换 |
| 人类行为 FCA | CAL — 生命因果架构 | 历史名称已替换 |
| AFA | CAL | 仅保留为历史标识 |
| TMA | CAL | 仅保留为历史标识 |
| Triplex Modulation Architecture | CAL | 仅保留为历史标识 |
| SN–DMN–FP–ECN 四系统结构 | 无公开的一一对应替代项 | 历史架构；非标准定义 |
| 激素调制层 | 不再作为独立 CAL 标准模块 | 历史结构；非标准定义 |
| 可观察输出分析入口 | 可观察输出 | 在更一般的因果层保留 |
| 反馈更新 | 现实反馈 → 结构更新 | 已一般化为公开定义 |

## 禁止一一变量迁移

FCA 向 CAL 的变化属于模型层级变化，不是术语替换。

不得把历史 FCA 系统机械改名为 CAL 变量。CAL 的受保护变量、形式化关系与 Runtime 程序不通过本迁移表公开。

## 现行使用规则

所有现行引用统一使用：

- CAL — Causal Architecture of Life
- CAL — 生命因果架构

历史 FCA 术语只用于说明仓库历史、既往发布内容或迁移来源。
