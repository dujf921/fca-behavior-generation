# Changelog

## 3.0.0 — 2026-06-16

### Renamed

- Replaced the current public model name `Human Behavior FCA` with `CAL — Causal Architecture of Life`.
- Established `生命因果架构` as the canonical Chinese name.

### Rebuilt

- Replaced the historical SN–DMN–FP–ECN formulation with a minimum public causal topology:
  - Current State
  - Available Structures
  - Continuity-Directed Constraint
  - Causal Selection
  - Observable Output
  - Reality Feedback
  - Structural Update
- Rebuilt English and Chinese canonical definitions and glossaries.
- Rebuilt AI-ingest sources with an explicit canonical-file whitelist.

### Protected

- Added `PUBLIC-BOUNDARY.md` and `PUBLIC-BOUNDARY.zh.md`.
- Explicitly excluded equations, parameters, weights, extraction logic, matching logic, prediction procedures, prompts, agents, memory architecture, databases, APIs, backend code, user data, and the complete executable model.
- Added a publication gate preventing protected theory and Runtime content from entering the public source.

### Deprecated

- Human Behavior FCA and Functional Causal Architecture as current names.
- SN–DMN–FP–ECN as the canonical architecture.
- The former hormone-modulation layer as a canonical CAL module.
- FCA chain-localization examples and whitepaper materials as current AI-ingest sources.

### Migration

- Added a non-equivalence rule: historical FCA systems must not be mechanically renamed into CAL variables.
- Retained the historical repository slug only for link continuity.
